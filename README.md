# mi-taller

## What uses

Expo https://docs.expo.io/guides/
Turtle-cli (local build) https://docs.expo.io/distribution/turtle-cli/
## Commands

Init:

```expo start```


Local build:

```turtle build:android -u 'EXPO_USER' -p 'EXPO_PASSWORD' -t apk```

Install on device:

```adb install PATH_TO_APK```
